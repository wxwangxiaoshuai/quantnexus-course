# 因子检验小项目

> 预计 **90 分钟** · 难度 **高级** · 选修模块 M10 · 无专用互动

## 背景

因子研究回答一个问题：**这个信号对未来收益有没有排序能力？** IC（信息系数）与分层回测是标准检验工具。

## 数据范围（必须明确）

| 项 | 最低要求 |
|----|----------|
| Universe | ≥4 个期货品种（建议：rb, hc, i, j 或课内 8 品种子集） |
| 区间 | ≥60 个交易日 |
| 频率 | 日频 |
| 因子 | 动量 **或** Carry（二选一） |
