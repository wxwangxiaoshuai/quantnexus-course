为什么？你马上就会看到：
- 均线 = 价格序列的**均值**
- 布林带 = 均线 ± k 个**标准差**
- 夏普比率 = 超额收益 / **标准差**（波动率）
- RSI = 涨跌幅的相对比例（**比值**）
- 正态分布 = 回测陷阱中「极端事件被低估」的元凶

如果你不理解均值、方差、标准差、分布——你看到的将是一堆神秘的公式，而永远无法理解指标背后的数学直觉。这一节，我们用最简单的方式建立这个直觉。

## 1. 收益率：量化策略的「基本粒子」

物理学有基本粒子（电子、夸克），量化交易的基本粒子是**收益率（Return）**。所有的 Sharpe、MDD、Sortino 都是从收益率推导出来的。

**简单收益率**（Simple Return）：

$$
r_t^{\text{simple}} = \frac{P_t}{P_{t-1}} - 1
$$

比如昨天螺纹钢 3500，今天 3570：$r = 3570/3500 - 1 = +2\%$。直觉上很直观对吧？但问题来了——从 3500 涨 2% 到 3570，再从 3570 跌 2% 到 3498.6，你没有回到原来的 3500。这是因为 $+2\%$ 和 $-2\%$ 不以相同的基础计算。

**对数收益率**（Log Return）：

$$
r_t^{\text{log}} = \ln\left(\frac{P_t}{P_{t-1}}\right)
$$

涨 2% 然后跌 2%，用对数收益率是 $+0.0198 + (-0.0202) = -0.0004$ ——相当于微跌了。对数收益率有**可加性**：多期收益率 = 各期收益率之和。这对多日回测的复利计算很重要。

> 日常直觉用简单收益率就够了。但做统计推论（假设检验、置信区间）时对数收益率更精确，因为它的分布更接近正态。

## 2. 均值：市场的「重心」

$$
\bar{r} = \frac{1}{N} \sum_{t=1}^{N} r_t
$$

均值的直觉：把 N 天的收益率加起来除以 N，就是这段时间的「平均日收益」。如果 $\bar{r} > 0$，说明这段时间总体是赚钱的；$\bar{r} < 0$ 则赔钱。

但均值只告诉你一个数字，没有告诉你赚钱和亏钱的日子有多分散——某天赚 5%、另一天亏 4%，均摊下来日均 +0.5%——听起来不错？等一下。只看均值是最容易犯的错误。你还需要了解分散度。

## 3. 方差与标准差：收益的「摇摆幅度」

**方差**衡量的是「收益率在均值周围散得多开」：

$$
\sigma^2 = \frac{1}{N-1} \sum_{t=1}^{N} (r_t - \bar{r})^2
$$

**标准差**是方差的平方根，它和收益率有相同的单位（百分比）：

$$
\sigma = \sqrt{\sigma^2}
$$

**直观理解**：
- 策略 A：每天收益在 +1% 和 -0.5% 之间小波动 → 标准差低（~0.5%），资金曲线平滑上升
- 策略 B：某天 +5%，某天 -4%，某天 +2%，某天 -3% → 标准差高（~3%），资金曲线大起大落

策略 A 的年化波动率 ≈ 0.5% × √252 ≈ 7.9%
策略 B 的年化波动率 ≈ 3% × √252 ≈ 47.5%

年化公式里的 √252 怎么来的？前提是「日收益率互不相关（独立）」，则方差随时间线性累加（N 天的方差 = 1 天方差 × N），所以 N 天的标准差 = 1 天标准差 × √N。取 N=252（一年约 252 个交易日），就得到年化波动率。

> 注意：日收益率不总是独立的（波动率聚集现象——大波动往往扎堆出现），所以 √252 只是一个近似。这也是风险管理中需要警惕的。

## 4. 正态分布：你被华尔街骗了（但还是要学）

大多数金融教材告诉你：「资产收益率服从正态分布」。它的形状像一个钟形曲线（Bell Curve）：大部分收益率集中在均值附近，±1 标准差覆盖约 68% 的日子，±2 标准差覆盖约 95%。

**问题是——金融市场收益率的实际分布不是正态的。**

现实中的收益率分布有三个关键特征：

**① 尖峰（Leptokurtosis）**：极端值比正态分布预测的多。正态分布认为 ±5 倍标准差事件「十亿年一遇」，但实际上金融市场每 2-3 年就会出现一次。2020 年新冠恐慌中原油价格跌到「负值」就是例证。

**② 肥尾（Fat Tail）**：那些「不可能发生」的极端事件，发生概率远高于正态分布预测。3σ 事件（超过 3 倍标准差的波动）在正态分布中概率约 0.3%，在真实市场中可能是这个数的 10-50 倍。

**③ 负偏（Negative Skewness）**：大跌比大涨更频繁且幅度更大。正态分布是对称的，但金融市场喜欢「慢涨急跌」——价格用几个月爬上去，用几天摔下来。

> 对量化策略的影响：如果你基于正态分布假设来设计仓位和止损（如假设 3σ 事件基本不会发生），你的策略会被肥尾打穿。这也是为什么我们在模块 7 中强调「压力测试」和「尾部风险对冲」——你需要为比正态分布预测更坏的情况做好准备。

## 5. 相关性：两个东西一起动的程度

**相关系数（Correlation Coefficient, ρ）**衡量的是两个序列（比如螺纹钢和热卷的日收益率）之间共变的程度。范围在 -1 到 +1 之间：

- ρ = +1：完全正相关（一个涨另一个也涨，且幅度完全一致）
- ρ = 0：不相关（各自独立涨跌）
- ρ = -1：完全负相关（一个涨一个跌，完美对冲）

$$
\rho_{X,Y} = \frac{\sum (X_i - \bar{X})(Y_i - \bar{Y})}{\sqrt{\sum (X_i - \bar{X})^2} \cdot \sqrt{\sum (Y_i - \bar{Y})^2}}
$$

在量化交易中，相关性有两大重要含义：

**1. 分散化的数学依据**：如果你持有相关性低于 0.5 的多个品种，组合波动率会低于单品种加权平均。这就是「不要把鸡蛋放在一个篮子里」的数学表达——我们在模块 8 会详细展开。

**2. 相关性 ≠ 因果性**：「过去两个月螺纹钢和豆粕的日收益率相关性为 0.7」≠「螺纹钢的涨跌直接导致豆粕涨跌」。可能只是刚好两个品种都处于类似的宏观环境中（如宽松货币 + 房地产刺激同时刺激了建材和农产品饲料需求）。用相关关系推导因果关系是量化研究中最常见的逻辑谬误。

> 相关性还有时间维度——牛市中的相关性往往比熊市中高（所谓的'相关性在危机中趋近于 1'）。当市场恐慌时，所有风险资产同时下跌，你的「分散化组合」可能瞬间失效。

```python
import math

def simple_return(prices):
    """简单收益率列表"""
    return [prices[i] / prices[i-1] - 1 for i in range(1, len(prices))]

def log_return(prices):
    """对数收益率列表"""
    return [math.log(prices[i] / prices[i-1]) for i in range(1, len(prices))]

def mean(data):
    """均值"""
    return sum(data) / len(data)

def std(data):
    """标准差（样本标准差，使用 N-1）"""
    m = mean(data)
    variance = sum((x - m) ** 2 for x in data) / (len(data) - 1)
    return math.sqrt(variance)

def annualized_vol(daily_returns):
    """年化波动率 = 日标准差 × √252"""
    return std(daily_returns) * math.sqrt(252)

# 案例：两个策略的对比
returns_a = [0.005, -0.003, 0.008, -0.002, 0.004, 0.001, -0.001, 0.006]
returns_b = [0.05, -0.04, 0.03, -0.03, 0.02, -0.01, -0.02, 0.04]

print(f"策略 A — 日均收益: {mean(returns_a):.2%}, 波动率: {annualized_vol(returns_a):.1%}")
print(f"策略 B — 日均收益: {mean(returns_b):.2%}, 波动率: {annualized_vol(returns_b):.1%}")
# 策略 A: 低波动、平稳
# 策略 B: 高波动、像坐过山车
```

:::highlight green
学完这一节的收获：你看布林带的时候不再想「为什么是均线 ± 2 倍标准差」，而是想「标准差衡量的是价格的分散程度，2σ 覆盖了正常波动范围，突破 2σ 意味着价格偏离了正常轨道」。你看夏普比率的时候不再想「1.5 这个数字是什么意思」，而是想「单位波动率创造的超额收益是平均日波幅的 1.5 倍」。数学从神秘符号变成了你理解市场的直觉工具。
:::
