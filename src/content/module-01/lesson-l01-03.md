期货最重要的特性之一：**可以先卖后买**（做空）。

股票买入后才能卖出。但期货不一样——你可以先"卖出开仓"，约定未来以某价格卖出，如果价格下跌再"买入平仓"，低买高卖，赚中间差价。

**关键区别（别记错）**：期货做空**不需要向任何人借入标的**。卖出开仓本身就是签下一份「未来以今天价格卖货」的合约，多空天然对称——这和股票的「融券做空」（要先借股票、还有借券成本和可借数量限制）机制完全不同。

**程序员类比**：做空 = 先签一份"未来以今天的价格卖出一批货"的合同，之后价格跌了再低价买货交割，赚差价。就像你先接下一个「按今天报价交付」的订单，等上游降价后再采购履约，赚中间的差价——你从头到尾没有借任何东西。

:::highlight green
做多：预期价格上涨 → 买入开仓 → 价格涨后卖出平仓 → 盈利
做空：预期价格下跌 → 卖出开仓 → 价格跌后买入平仓 → 盈利
:::

**保证金与杠杆**：交易期货不需要付全款，只需要缴纳合约价值 5%~15% 的保证金就可以控制整手合约。这就是杠杆——用小钱撬动大仓位，盈亏也同比放大。

这是把双刃剑：10 倍杠杆下，标的物涨 10%，你赚 100%；但跌 10%，你亏光本金。

```typescript
// 做多 vs 做空盈亏计算函数
function calculatePnL(
  direction: 'long' | 'short',
  entryPrice: number,
  exitPrice: number,
  lots: number,
  multiplier: number,
  marginRate: number,
): { profit: number; returnRate: number } {
  const priceDiff = exitPrice - entryPrice;
  // 做多：涨赚钱，跌亏钱；做空：跌赚钱，涨亏钱
  const sign = direction === 'long' ? 1 : -1;
  const profit = sign * priceDiff * lots * multiplier;
  const margin = entryPrice * lots * multiplier * marginRate;
  return { profit, returnRate: profit / margin };
}

// 案例：做空螺纹钢
// 判断要跌，在 3500 卖出开仓 1 手，3400 买入平仓
const result = calculatePnL('short', 3500, 3400, 1, 10, 0.10);
// 做空：(3400-3500) = -100 点，sign=-1，所以 profit = -1*(-100)*1*10 = +1000 元
// 收益率 = 1000 / (3500*1*10*0.10) = 1000/3500 ≈ +28.6%
```

:::highlight green
期货 vs 股票核心对比：

| 维度 | 股票 | 期货 |
|------|------|------|
| 交易方向 | 只能先买后卖（做多） | 可做多也可做空 |
| T+制度 | T+1（当天买入次日才能卖） | T+0（当天可多次买卖） |
| 杠杆 | 1:1（100%自有资金，融资除外） | 5-20 倍（保证金交易） |
| 到期日 | 无到期（只要不退市） | 有固定的到期日（交割月） |
| 持仓成本 | 无持有成本（融资有息费） | 有换月成本（展期） |
| 结算 | 浮盈浮亏，卖出时结算 | 逐日盯市，每天强制结算 |
| 交易时间 | 9:30-15:00 | 日盘+夜盘（部分品种），更长 |
| 数据粒度 | 通常到秒/分钟 | Tick 级别高频数据 |
:::

**强平（强制平仓）**：当你的账户「可用资金」降为负数（即保证金不够覆盖持仓风险了），经纪商会通知你补充保证金。如果你没有在规定时间内补足，经纪商有权强制平掉你的持仓——不管你在不在、愿不愿意。

> 程序员视角：强平就像 Kubernetes 的 OOM Killer——当内存（保证金）不够用，系统直接 kill 容器（持仓），不会跟你商量。你的量化系统必须有「账户健康度」监控模块，在接近强平线之前主动减仓，而不是依赖经纪商的被动通知。

**一个小案例帮你记住「为什么做空可能赚得比做多快」**：螺纹钢从 3000 涨到 6000 需要翻倍，但从 6000 腰斩到 3000 只需要跌 50%——涨幅需要更大的绝对价格变动才能实现相同的收益率。在趋势性下跌中（如 2021 年铁矿石从 1300 跌到 600），做空策略往往表现亮眼。但做空的理论亏损是无限的（价格可以无限涨），做多的理论亏损是有限的（最多跌到 0），这个不对称性是风控系统必须处理的。
