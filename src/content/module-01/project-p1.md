# 杠杆与保证金实战

> 预计 **30 分钟** · 难度 **入门** · 核心互动：杠杆计算器

## 背景

期货是保证金交易：你用少量本金控制更大名义价值的合约。杠杆放大了收益，也放大了亏损——理解「保证金率 → 强平」的链条，是后续所有仓位管理的基础。

## 实验步骤

### 步骤 1：打开互动实验

前往课节 [L01-02 合约要素详解](/curriculum/1/L01-02)，找到 **杠杆计算器** 互动组件。

### 步骤 2：完成情景推演

至少填写 **4 组** 不同参数，建议覆盖：

| 组别 | 建议本金 | 建议杠杆 | 价格变化 |
|------|----------|----------|----------|
| A | 10 万 | 10× | +3% |
| B | 10 万 | 10× | -3% |
| C | 10 万 | 20× | -2% |
| D | 5 万 | 5× | -10% |

记录每组输出的 **保证金率** 与 **是否触发强平**。

### 步骤 3：撰写对照表

将结果整理为下表（可手写或 Markdown）：

| 本金 | 杠杆 | 价格变化 | 保证金率 | 是否强平 | 备注 |
|------|------|----------|----------|----------|------|
| | | | | | |

### 步骤 4：强平说明

用 **200 字以内** 回答：为什么 C 组（高杠杆 + 小幅反向）比 D 组更容易强平？

## 可选拓展

- 阅读 [L01-03 做多与做空](/curriculum/1/L01-03)，说明空头在价格上涨时保证金率如何变化
- 思考：计算器中的强平阈值是教学简化，实盘还需考虑维持保证金与交易所规则

## 反思

1. 若单笔最多愿意亏损本金的 2%，对应的最大合理杠杆大约是多少？
2. 「穿仓」与「强平」有何区别？

:::highlight blue
**说明**：本实验为教学简化模型，不构成投资建议。实盘保证金规则因品种与券商而异。
