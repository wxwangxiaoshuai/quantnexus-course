前面讲了三种仓位法，但到底「多大多小」才是最优？凯利公式（Kelly Criterion）给出了数学上的答案。它告诉你：在一场已知赔率和胜率的赌局中，每次应该下注总资金的多少比例，才能让长期增长率最大化。

## 凯利公式推导

假设你参与一个赌局：
- 赢的概率为 $p$（胜率）
- 赢时获得 $b$ 倍本金（赔率，不含本金）
- 输时损失投入的本金

**问题**：每次下注总资金的比例 $f$ 应该是多少？

**凯利公式**：

$$
f^* = \frac{pb - (1-p)}{b} = \frac{p \cdot (b+1) - 1}{b}
$$

注意：上式中 $\frac{pb-(1-p)}{b}$ 与 $\frac{p(b+1)-1}{b}$ 是**同一公式的代数变形**（把 $pb-(1-p)=p(b+1)-1$ 代入即得），不是两个不同公式。把它进一步拆开还能写成 $f^* = p - \frac{1-p}{b}$，仍是同一个式子。

**真正「另一种形式」——当赔率是「含本金」的总赔率** $b'$（赢时账户变为本金的 $b'$ 倍，$b' = b+1$）时，公式应写为：

$$
f^* = \frac{p \cdot b' - 1}{b' - 1}
$$

区别只在于 $b$ 用的是**净赔率**还是**含本金总赔率**，代入前务必分清，否则仓位会算错。

**公式含义**：
- 分子 $pb - (1-p)$ = 每单位本金的期望收益。只有期望收益 > 0（有正优势）时 $f^* > 0$
- $f^*$ 是「最优比例」——低于它增长太慢，高于它反而可能破产（波动过大侵蚀几何收益）

**直观理解**：一个胜率 60%、盈亏比 = 1（赢了赚 100 输了亏 100）的赌局：$f^* = 0.6 - 0.4/1 = 0.2$。最优每次下注 20%。如果下注 40%，长期增长率反而低于 20%——这就是「过度下注」的陷阱。

```python
def kelly_formula(win_rate, avg_win, avg_loss):
    """
    凯利公式计算最优仓位比例
    win_rate: 胜率 (0-1)
    avg_win: 平均每笔盈利（正数）
    avg_loss: 平均每笔亏损（正数，取绝对值）
    返回: 最优仓位比例
    """
    b = avg_win / avg_loss  # 盈亏比
    f_star = (win_rate * b - (1 - win_rate)) / b
    return max(0, f_star)  # 如果无优势则返回 0

# 实战案例
def kelly_example():
    # 案例1：胜率 40%，盈亏比 3:1
    f1 = kelly_formula(0.4, 300, 100)
    print(f"胜率40% 盈亏比3:1 → 最优仓位: {f1:.1%}")  # 20%
    
    # 案例2：胜率 60%，盈亏比 1:1
    f2 = kelly_formula(0.6, 100, 100)
    print(f"胜率60% 盈亏比1:1 → 最优仓位: {f2:.1%}")  # 20%
    
    # 案例3：胜率 30%，盈亏比 5:1 (趋势跟踪)
    f3 = kelly_formula(0.3, 500, 100)
    print(f"胜率30% 盈亏比5:1 → 最优仓位: {f3:.1%}")  # 16%
```

:::highlight blue
凯利公式的核心洞见：低胜率高盈亏比的策略（如趋势跟踪，胜率30%盈利5倍）和 高胜率低盈亏比的策略（如均值回归，胜率60%盈利等额），都可能有接近的最优仓位。策略的好坏不由胜率决定，而由「胜率 × 盈亏比」这一对数字的综合效应决定。
:::

## 半凯利（Half-Kelly）与分数凯利

实际交易中，很少有人按完全凯利下注，原因有三：

1. **参数不确定**：你的胜率和盈亏比是从历史回测中估算的，有采样误差。凯利给出的 $f^*$ 基于完美信息，现实中你永远不知道真实胜率。
2. **波动承受**：完全凯利的资金曲线波动极大，日波动可达 20%。绝大多数人在回撤 30% 时就会放弃策略，导致无法享受长期复利。
3. **连亏序列**：即使最优 $f^*$ = 20%，连续亏损 5 次就损失 67%，10 次就只剩 10%。凯利不考虑你是否有心理承受能力。

**实用建议**：
- **半凯利（Half-Kelly, $f^* / 2$）**：回撤减半，增长率只损失 25%。是业界最常用的折中方案。
- **四分之一凯利**：更保守，适合机构或低风险偏好。
- **经验法则**：个人交易者，$f^* < 2%$ 太胆小，$f^* > 10%$ 太大胆。2%-5% 是大多数人能承受的区间。

## 固定分数法 vs 凯利法

| 方法 | 公式 | 优点 | 缺点 |
|------|------|------|------|
| 固定分数（Fixed Fraction） | 每次用总资产 $k\%$ 开仓 | 简单、防破产 | 不考虑策略质量差异 |
| 凯利（Kelly） | $f^* = p - (1-p)/b$ | 数学最优、最大化长期增长率 | 参数不确定、波动过大 |
| 半凯利（Half-Kelly） | $f^* / 2$ | 保留大部分增长率，大幅降低波动 | 不是绝对的「最优」 |
| 最优 f（Optimal f） | 最大化几何增长率 | 理论最优 | 需要完整盈亏分布，极度敏感 |
