# 破产与仓位模拟

> 预计 **45 分钟** · 难度 **进阶** · 核心互动：破产模拟器（风险比例 × 蒙特卡洛）

## 背景

单笔风险比例（每笔愿意亏掉的权益百分比）决定了长期 **生存概率**。本互动用蒙特卡洛模拟不同风险比例下的权益路径；**凯利公式** 的理论最优比例见 L07-03（可选对照）。

## 实验步骤

### 步骤 1：运行模拟

打开 [L07-01 仓位管理](/curriculum/7/L07-01) 中的 **破产模拟器**。

固定胜率与盈亏比（或使用默认），分别测试 **≥3 种** 单笔风险比例，例如：

| 风险比例 | 100 次模拟破产率（约） | 典型最终权益区间 |
|----------|------------------------|------------------|
| 2% | | |
| 5% | | |
| 10% | | |
| 20%（可选） | | |

### 步骤 2：写出仓位红线

根据上表，写一条 **可执行的规则**，例如：

> 当单笔风险 > ___% 且模拟破产率 > ___% 时，必须降仓至 ___% 以下。

### 步骤 3（可选）：对照凯利

阅读 [L07-03 凯利公式](/curriculum/7/L07-03)，说明：为何实盘中常用 **半凯利或更低**，而非全凯利？

## 反思

1. 模拟器假设每笔盈亏独立——真实序列相关时，破产率会更高还是更低？
2. 「永不破产」是否等于「长期盈利」？
