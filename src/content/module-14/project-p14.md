# 多因子截面策略

> 预计 **5 小时**（含实操） · 难度 **专家** · 课节驱动 · 结业项目

## 定位说明

M14 是课程 **结业项目**：8 品种因子库 → 截面调仓 → 三策略组合 → 运维下架。五（四）节项目课承载全部实操；本页为 **总览与验收**。

:::highlight amber
**前置建议**：L14-01 标注先修 [L10-02 IC 与分层回测](/curriculum/10/L10-02)。若未学 M10，请先完成该课或课内 IC 速成段落。
:::

:::highlight amber
**诚实边界**：课内 `CrossSectionalMomentumStrategy` 为 **单品种骨架**；真正截面组合需多品种引擎。组合章节的示例业绩表为 **教学演示**，验收须提交 **你自己** 的回测结果或明确标注的假设情景。
:::

## 验收清单

- [ ] L14-01～L14-04 全部课节标记完成
- [ ] **因子库**：动量 + 波动率（或 Carry）至少 2 因子，附 IC 表
- [ ] **截面回测**：Weekly Top/Bottom 或课内等价流程，附收益曲线或指标
- [ ] **组合分配**：基于 M12/M13/M14 三策略 **自测指标** 的 ERC 权重（禁止照搬 L14-03 示例表）
- [ ] **运维**：监控指标 ≥3 项；下架标准 ≥2 条可量化规则

## L14-03 特别要求

课内示例表（策略 A/B/C 的 Sharpe、回撤）仅用于演示 ERC 公式。你的交付须：

1. 替换为 **你在 P12/P13/P14 中的实际或假设指标**，或
2. 在表格标题注明「**假设情景，非回测结果**」

## 反思

三个策略低相关时，组合 Sharpe 为何可能高于任一单策略？
