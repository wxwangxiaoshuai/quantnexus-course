## 离线因子 IC

将 ≥5（建议 8）个品种日线 CSV 放入同一目录：

```bash
python scripts/offline_capstone/m14_factor_ic.py --dir ./data/symbols
```

写入 IC 均值、t-stat、胜率（|t|>2 门槛与课内一致）。
