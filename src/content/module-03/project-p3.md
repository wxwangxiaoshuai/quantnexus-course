# Tick 聚合实验

> 预计 **40 分钟** · 难度 **入门** · 核心互动：Tick→K 线

## 背景

行情的最细粒度是 **Tick**（逐笔成交）。K 线是对 Tick 在固定时间窗口内的 **OHLCV 聚合**。本实验聚焦聚合逻辑；主力合约拼接见 L03-02（可选拓展）。

## 实验步骤

### 步骤 1：运行聚合互动

前往 [L03-01 Tick 数据](/curriculum/3/L03-01)，打开 **Tick→K 线** 互动。

### 步骤 2：观察窗口边界

1. 播放 Tick 流，观察 K 线如何随时间 **逐根闭合**
2. 切换至少 **2 种** 聚合节奏，对比同一时段 K 线数量差异
3. 指出「当前未闭合 K 线」的 Close 为何会变动

### 步骤 3：字段释义

任选 **1 根已闭合 K 线**，解释五个字段：

| 字段 | 含义 | 本根 K 线的值/说明 |
|------|------|-------------------|
| Open | 窗口内第一笔成交价 | |
| High | 窗口内最高成交价 | |
| Low | 窗口内最低成交价 | |
| Close | 窗口内最后一笔成交价 | |
| Volume | 窗口内成交量累计 | |
