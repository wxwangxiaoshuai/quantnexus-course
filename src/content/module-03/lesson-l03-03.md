在前面的课程中，你学习了 Tick 数据和 K 线聚合。现在把视角从「原始数据」转向「数据的统计特征」——这是量化策略开发中最重要的探索性数据分析（EDA）环节。

**为什么要关心收益率的分布？**

想象你收到了两个策略的回测报告：
- 策略 A：年化收益 30%，最大回撤 15%
- 策略 B：年化收益 30%，最大回撤 15%

两者数字一模一样。但策略 A 的每日损益像小溪流水一样平缓（+0.1%、-0.05%、+0.2%……），而策略 B 的每日损益像过山车（+5%、-4%、+2%、-6%、+8%……）。

均值相同、回撤相同——但两个策略的风险特征完全不同。如果你只看均值，就犯了量化交易中最经典的错误：**忽略分布的形状**。

```python
import numpy as np
from scipy import stats

def compute_log_returns(prices):
    """对数收益率: r_t = ln(P_t / P_{t-1})"""
    return np.diff(np.log(prices))

def analyze_returns(returns):
    """对收益率序列做完整的统计特征分析"""
    mean_daily = np.mean(returns)
    mean_annual = mean_daily * 252
    std_daily = np.std(returns, ddof=1)
    vol_annual = std_daily * np.sqrt(252)
    # 第三矩：偏度——分布的不对称性
    skew = stats.skew(returns)
    # 第四矩：超额峰度——尾部厚度
    kurt = stats.kurtosis(returns)  # scipy 默认返回超额峰度
    sharpe = mean_annual / vol_annual if vol_annual > 0 else 0
    print(f"年化收益率: {mean_annual:.2%}")
    print(f"年化波动率: {vol_annual:.2%}")
    print(f"偏度 (Skewness): {skew:.3f} {'←负偏，跌比涨猛' if skew < -0.5 else ''}")
    print(f"超额峰度: {kurt:.3f} {'←肥尾风险!' if kurt > 2 else ''}")
    print(f"夏普比率: {sharpe:.2f}")
    if skew < -0.5 and kurt > 2:
        print("\n⚠ 偏度为负且峰度高：策略有'慢涨急跌'的风险特征")
        print("建议：不要仅靠夏普比率评估策略，关注最大回撤和尾部VaR")
    return {'annual_return': mean_annual, 'annual_vol': vol_annual, 'skewness': skew, 'sharpe': sharpe}
```

**统计矩——收益率分布的「四维指纹」**

统计学中用「矩」来描述分布的形状。四个最重要的矩：

**第一矩 — 均值**：你的策略平均每天赚（或亏）多少钱。均值为正 ≠ 策略能赚钱——因为均值可能被几个极端收益日拉高。

**第二矩 — 标准差**：收益率在均值周围的分散程度。标准差大 = 每日收益波动大 = 资金曲线起伏剧烈。

**第三矩 — 偏度（Skewness）**：分布是「左偏」还是「右偏」。偏度为负：大亏比大赚更频繁——如卖保险策略（日常赚小钱，偶尔一个大亏）。偏度为正：大赚比大亏更频繁——如强趋势跟踪策略。

**第四矩 — 峰度（Kurtosis）**：分布的「峰值」和「尾部厚度」。统计学常用**超额峰度**（excess kurtosis = 峰度 − 3），正态分布的超额峰度 = 0。超额峰度 > 0 → 策略遭遇极端收益的概率比正态分布预测的高——即 **「肥尾」**。代码中 `scipy.stats.kurtosis` 默认返回超额峰度。

:::highlight orange
「华尔街的物理嫉妒」：金融学对正态分布的执念源于它数学上的便利——只需要均值和方差两个参数就能完整描述。但现实是：金融市场收益率是尖峰肥尾的，±5σ 事件在正态分布中「永远不会发生」，在真实市场中每 2-3 年就上演一次。2020 年 WTI 原油跌到 -37 美元/桶，正态分布给出的概率约 10^{-50}——但实际发生了。
:::

**波动率聚集（Volatility Clustering）——「大波动后面还是大波动」**

如果你把螺纹钢每日收益率画出来，一眼就能看到：波动大的日子喜欢扎堆出现——连续几天大幅涨跌后，接下来几天也是大幅波动。

**直觉理解**：市场像一个被点燃的火药桶。第一声巨响不仅自己很大声，还会引发连锁爆炸（止损盘、恐慌盘、杠杆爆仓）。这些连锁反应在几天内陆续释放，所以「连续几天波动都很大」。等连锁反应全部释放完，市场回归平静——直到下一个火花。

**对策略的实战意义**：大波动来临时止损不应太紧；高波动环境下降低仓位；波动率可作为策略的「暂停开关」——当日波动率超过历史 95% 分位时暂停开新仓。

:::highlight green
量化交易的第一条纪律：用数据定义风险，而不是用直觉。均值告诉你「平均能赚多少」，但偏度和峰度告诉你「亏的时候会亏成什么样」。一个只看均值的量化交易者，就像只看动力不检查刹车的赛车手——第一次出事故之前一切都很完美。
:::
