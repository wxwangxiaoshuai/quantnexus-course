# 双均线策略沙盒

> 预计 **60 分钟** · 难度 **进阶** · 分两部分：沙盒推演 + 代码关卡

## 背景

双均线策略 = 快线上穿慢线做多、下穿平仓。本项目的 **两个交付物对应两个不同工具**：

1. **L05-01 策略沙盒**：调均线周期、单步看信号（不能改写开仓逻辑）
2. **L05-03 代码关卡**：改写 `shouldBuy` 开仓条件并通过测试

## Part A：沙盒推演（约 30 分钟）

### 步骤

1. 打开 [L05-01 策略纯函数](/curriculum/5/L05-01) 中的 **策略沙盒**
2. 使用 **单步执行**，观察金叉/死叉时持仓如何变化
3. 对比至少 **2 组** 均线周期（如 5/20 vs 10/30），记录资金曲线差异

### 记录模板

| 周期组 | 交易次数（约） | 曲线特征 | 备注 |
|--------|----------------|----------|------|
| 5/20 | | | |
| 10/30 | | | |

## Part B：改写开仓条件（约 30 分钟）

### 步骤

1. 打开 [L05-03 代码关卡](/curriculum/5/L05-03) 的 **CodeChallenge**
2. 在 `shouldBuy` 中实现一版 **自定义开仓逻辑**（例如在双均线基础上增加过滤条件）
3. **通过全部测试用例**

### 提交说明

无需外部 Git 提交——**通过 CodeChallenge 即视为「提交成功」**。在笔记中写 3 句话说明：你的改写与默认双均线有何不同。

## 反思

1. 为何「全路径推演」指的是在沙盒中逐步执行到样本结束，而非跳步浏览？
2. 改写开仓条件时，如何避免引入前视偏差？
