单一策略面对的市场环境是有限的。牛市中的趋势跟踪表现好，熊市可能中等，震荡市大概率亏。但如果你同时运行一个趋势跟踪策略和一个均值回归策略——一个在趋势市中赚钱、一个在震荡市中赚钱——你的整体资金曲线会平滑得多。这就是多策略组合的核心价值。

## 多策略组合的核心公式

如果两个策略的日收益相关性为 $\rho$，组合后的波动率为：

$$
\sigma_{\text{portfolio}} = \sqrt{w_1^2\sigma_1^2 + w_2^2\sigma_2^2 + 2w_1w_2\rho\sigma_1\sigma_2}
$$

其中 $w_1, w_2$ 为资金权重，$\sigma_1, \sigma_2$ 为各自的波动率。

**关键洞察**：当 $\rho < 1$（策略不完全同涨同跌），组合的波动率 < 加权平均波动率。这就是「分散化效应」的数学原理。$\rho$ 越低，分散效果越好。

## 策略组合的三种方式

**1. 等权组合**：
- 每个策略分配相等的资金。最简单、不易过拟合。
- 缺点：忽略了策略质量的差异（好的和差的各拿一样多）。

**2. 波动率加权（Risk Parity）**：
- 每个策略分配资金使其对组合的风险贡献相等。
- 高波动策略少分配，低波动策略多分配。
- 公式：$w_i = \frac{1/\sigma_i}{\sum_j 1/\sigma_j}$

**3. 凯利组合**：
- 用多个策略的历史盈亏分布计算每个策略的凯利比例，按比例分配。
- 理论最优，但对输入参数极度敏感。

```python
def risk_parity_weight(strategy_sharpes, strategy_vols, correlations):
    """
    简单风险平价：按策略波动率的倒数分配权重
    strategy_vols: dict, key=策略名, value=年化波动率
    """
    inv_vols = {k: 1.0 / v for k, v in strategy_vols.items()}
    total = sum(inv_vols.values())
    weights = {k: v / total for k, v in inv_vols.items()}
    return weights

# 多策略组合回测
def portfolio_backtest(strategies, weights, bars):
    """
    strategies: dict, key=name, value=日收益列表
    weights: dict, key=name, value=资金权重
    """
    daily_returns = []
    n = len(next(iter(strategies.values())))
    for i in range(n):
        port_ret = sum(weights[name] * strategies[name][i] 
                       for name in strategies)
        daily_returns.append(port_ret)
    
    # 计算组合指标
    avg = sum(daily_returns) / n
    std = (sum((r - avg) ** 2 for r in daily_returns) / (n - 1)) ** 0.5
    sharpe = avg / std * (252 ** 0.5) if std > 0 else 0
    
    return {'returns': daily_returns, 'sharpe': sharpe, 'volatility': std}
```

:::highlight blue
多策略组合的目标不是「每个策略都赚钱」，而是「组合整体在任何市场环境下都能活下去」。一个好的组合中，A 策略亏钱的时候 B 策略大概率赚钱，反之亦然。找到这样的「互补策略对」比找到两个都赚钱的策略更难，也更有价值。
:::

## 因子研究入门：策略的「元语言」

你前面学的趋势跟踪、均值回归、动量——在专业量化团队的语言中，它们都可以被重新表述为**因子（Factors）**。因子是对「什么驱动了收益率」的可量化表达。理解因子，是你从「会用策略」升级到「会设计策略」的关键跨度。

**什么是因子？**

因子 = 一个可计算的数值特征，用于预测某资产未来的相对表现。每个品种在每个交易日都有一个因子值，排序后可以跨品种对比谁更值得买（或卖）。

**四大经典因子（CTA 领域的基石）**

| 因子 | 计算方法 | 直觉 |
|------|---------|------|
| **动量（Momentum）** | 过去 N 个月收益率 | 涨的继续涨，跌的继续跌 |
| **展期收益（Carry/Roll Yield）** | 近月价格 vs 远月价格的年化收益 | 基差隐含的「持仓收益」——在模块 1 基差课时你已见过这个概念 |
| **价值（Value）** | 当前价格 vs 长期均值（如 5 年均价）的偏离 | 回归均值的力量——均值回归策略就是利用了这个因子 |
| **质量/波动率（Quality/Low Vol）** | 低波动品种的 Sharpe 往往更高 | 「安静的品种比躁动的品种更赚钱」的长期统计现象 |

**因子 vs 策略的关系**

策略（如双均线金叉）是对因子的**工具化实现**：
- 双均线金叉 = 用 MA 交叉来测量「趋势是否已经形成」= 动量因子的实现
- 布林带反转 = 用标准差来衡量「价格偏离均值多少」= 价值因子的实现
- 海龟策略 = Donchian 通道（动量） + ATR（波动率） = 动量+波动率双因子

> 理解因子之后，你的策略设计思维会从这个：「我想做一个好的双均线策略」升级到这个：「我想找到某个品种的动量因子是否有持续性，然后针对它设计一个进场出场逻辑」。你不再被「双均线好用吗」这类问题困扰，而是从第一性原理出发去思考。

**因子研究方法论（简化版）**

1. **定义因子**：写出因子的计算公式（过去 N 期收益率？展期收益率？波动率倒数？）
2. **计算因子值**：对每个品种每个交易日计算一个因子值
3. **检验预测力**：看当天因子值高的品种，下一期的收益率是否确实高于因子值低的品种（这叫 IC = Information Coefficient）
4. **转化为策略**：如果因子有预测力，设计一个买入/卖出规则来捕获这个规律
5. **控制风险**：因子集中的品种要做空头或减仓（避免因子暴露过度集中在一个板块）

> 因子研究是一个完整的领域，这里只是给你一个入门地图。当你觉得自己可以独立判断「这个因子值不值得交易」时，你就迈入了真正的策略研发阶段。
