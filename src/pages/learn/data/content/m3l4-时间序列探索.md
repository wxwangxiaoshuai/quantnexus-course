上一节学会了「看分布」——收益率长什么样、偏度是不是负的、有没有肥尾。这一节学「看时间轴」：昨天的收益和今天有关吗？今天是周五会不会不一样？

**时间序列分析 = 量化策略开发的「X 光检查」**。在做任何回测之前，先弄清楚你的数据在时间维度上有什么规律。

**自相关（Autocorrelation, ACF）——昨天会影响今天吗？**

自相关衡量的是「今天的收益率和 k 天前的收益率之间的关系」。

**三个真实例子**：
- **高频均值回复**：1 分钟螺纹钢收益率与 t-1 分钟有显著负相关（lag-1 < 0）——「涨了下一分钟倾向于跌」
- **趋势延续**：某些强趋势品种的日线 lag-1 自相关为正——「今天大涨、明天大概率继续涨」
- **白噪声**：完美随机的价格序列，所有 lag 自相关接近零

**数学定义**（lag-k 自相关系数）：

$$\rho_k = \frac{\text{Cov}(r_t, r_{t-k})}{\text{Var}(r_t)}$$

如果某个 lag 的自相关系数超过 ±1.96/√T（95% 置信区间），说明该滞后项与当前收益有统计显著的相关性。

```python
import numpy as np
from statsmodels.tsa.stattools import acf, pacf, adfuller

def time_series_diagnosis(returns, nlags=40):
    """对收益率序列做全面的时间序列诊断"""
    acf_values = acf(returns, nlags=nlags)
    ci = 1.96 / np.sqrt(len(returns))
    significant_lags = [lag for lag in range(1, nlags+1) if abs(acf_values[lag]) > ci]
    print(f"[ACF] 95%置信区间: ±{ci:.4f}")
    if significant_lags:
        print(f"显著自相关lag: {significant_lags}")
    else:
        print("无显著自相关")
    # 偏自相关
    pacf_values = pacf(returns, nlags=nlags)
    pacf_sig = [lag for lag in range(1, nlags+1) if abs(pacf_values[lag]) > ci]
    print(f"[PACF] 显著lag: {pacf_sig if pacf_sig else '无'}")
    # 平稳性检验
    adf_result = adfuller(returns, autolag='AIC')
    print(f"[ADF] p-value: {adf_result[1]:.6f} {'✅ 平稳' if adf_result[1] < 0.05 else '❌ 不平稳'}")
```

**偏自相关（PACF）——剔除中间变量后的「净关系」**

ACF 有一个问题：当 lag-3 自相关显著时，可能不是因为第三天真能预测今天，而是第三天通过第二天和第一天间接影响了今天。

PACF 解决了这个问题——它剔除中间变量影响，只看 lag-k 的「净效应」。

> ACF 问：「第三天和今天有关系吗（不管第二天）？」PACF 问：「在已知第二天的条件下，第三天还能提供什么？」

:::highlight blue
完全有效市场的收益率应该没有可预测的自相关——今天的涨跌不该泄露明天的方向。但现实中，微观结构（买卖价差、流动性冲击、信息扩散速度）导致短期自相关出现。高频策略吃的就是这笔微观结构留下的「剩饭」。对日频策略来说，自相关几乎是零——这是好事。太强的自相关意味着数据可能有问题（幸存者偏差、未来函数泄露）。
:::

**平稳性（Stationarity）——你的数据真的适合回测吗？**

一个时间序列是弱平稳的，如果均值、方差、协方差不随时间变化。

**为什么回测需要平稳性？** 你学到的统计规律（如「金叉后涨 2%」）只有在「明天和昨天统计特征一样」的前提下才有效。

**ADF 检验**：原假设是「序列有不平稳的单位根」，p-value < 0.05 说明平稳。
- 原始价格序列：几乎总是不平稳的（有趋势）
- 收益率序列：通常平稳 → **量化策略应该在收益率上做文章**

**日历效应（Calendar Effects）**

- **周一效应**：周末消息累积，周一开盘跳空大、波动高
- **周五效应**：短线交易者倾向周五平仓，尾盘有特殊的买卖压力
- **换月效应**：交割月前交易量急降、基差收敛——期货特有的陷阱

**危险场景**：你在 2022-2024 年数据上开发的策略，回测显示它经常「周一买、周五卖」——但这个规律可能只是那几年恰好如此。2025 年宏观变了，周一不再低开，策略失效——你完全不知道为什么。
