## 配对交易策略设计：价差 Z-score 的入场/出场逻辑

上一课我们确认了卷螺差适合配对交易。现在写策略代码。

**策略逻辑**：
1. 计算价差 = hc 价格 - rb 价格
2. 计算价差的滚动均值和标准差（60 日窗口）
3. Z-score = (价差 - 均值) / 标准差
4. Z-score > 2.0 → 做空 hc + 做多 rb（等价差缩小）
5. Z-score < -2.0 → 做多 hc + 做空 rb（等价差扩大）
6. Z-score 回归 ±0.5 → 平仓
7. Z-score > 3.0（继续扩大）→ 止损

**腿风险（Leg Risk）**：配对交易同时下两个单（rb 和 hc），可能一端成交一端不成交——这是套利策略特有的风险，需要在风控中处理。

> 关联课程：m5l4「策略类型」→ m8l1「套利范式」→ m10l1「因子思维」

```python
from vnpy.trader.utility import ArrayManager
from vnpy_ctastrategy import CtaTemplate
import numpy as np


class PairTradingStrategy(CtaTemplate):
    """
    卷螺差配对交易策略
    ⚠ vn.py CtaTemplate 绑定单合约，on_bar 只推送该合约，无法订阅 hc——
    本代码仅展示价差信号逻辑结构，实际配对交易走 m13l2 的离线价差回测（见下方脚本）。
    完整实盘版需 PortfolioStrategyTemplate 或多合约策略基类。
    逻辑：rb 和 hc 价差偏离均值时做多低估品种、做空高估品种
    注意：需要同时订阅 rb 和 hc 两个品种的行情
    """
    author = "QuantNexus-Student"
    
    spread_period = 60        # 价差均值/标准差计算周期
    entry_zscore = 2.0        # 入场 Z-score 阈值
    exit_zscore = 0.5         # 出场 Z-score 阈值
    stop_zscore = 3.0         # 止损 Z-score 阈值
    
    parameters = [
        "spread_period", "entry_zscore", "exit_zscore", "stop_zscore"
    ]
    variables = ["spread_value", "spread_mean", "spread_std", "zscore"]
    
    spread_value = 0.0
    spread_mean = 0.0
    spread_std = 0.0
    zscore = 0.0

    def __init__(self, cta_engine, strategy_name, vt_symbol, setting):
        super().__init__(cta_engine, strategy_name, vt_symbol, setting)
        self.am = ArrayManager(size=100)
        self.spread_history = []  # 价差历史序列

    def on_init(self):
        self.load_bar(100)

    def on_start(self):
        pass

    def on_stop(self):
        pass

    def on_bar(self, bar):
        self.cancel_all()
        self.am.update_bar(bar)
        if not self.am.inited:
            return
        
        # 计算价差（简化版：假设 hc = rb + 180）
        # 完整版需要从第二个品种获取实时价格
        rb_close = bar.close_price
        hc_close = rb_close + 180  # 历史均值
        spread = hc_close - rb_close
        self.spread_value = spread
        
        self.spread_history.append(spread)
        if len(self.spread_history) > self.spread_period:
            self.spread_history = self.spread_history[-self.spread_period:]
        
        if len(self.spread_history) >= self.spread_period:
            self.spread_mean = np.mean(self.spread_history)
            self.spread_std = np.std(self.spread_history, ddof=1)  # 与 m13l1 离线 df.std() 口径一致（样本标准差）
            if self.spread_std > 0:
                self.zscore = (spread - self.spread_mean) / self.spread_std
        
        # ---- 交易逻辑（本策略绑定 rb 单合约，只下 rb 腿；hc 腿需多合约模板）----
        # 价差 = hc - rb；zscore > 2 表示 hc 相对 rb 高估 → 做多 rb（价差回归时 rb 升/hc 降）
        if self.pos == 0:
            if self.zscore > self.entry_zscore:
                # 价差过大（hc 高估、rb 低估）→ 做多 rb
                self.buy(bar.close_price, 1)
            elif self.zscore < -self.entry_zscore:
                # 价差过小（hc 低估、rb 高估）→ 做空 rb
                self.short(bar.close_price, 1)
        
        elif self.pos > 0:
            if self.zscore < self.exit_zscore:
                self.sell(bar.close_price, abs(self.pos))
            elif self.zscore > self.stop_zscore:
                self.sell(bar.close_price, abs(self.pos))  # 止损
        
        elif self.pos < 0:
            if self.zscore > -self.exit_zscore:
                self.cover(bar.close_price, abs(self.pos))
            elif self.zscore < -self.stop_zscore:
                self.cover(bar.close_price, abs(self.pos))  # 止损
```

## 单品种回测 vs 配对回测的关键差异

| 差异维度 | 趋势跟踪（项目一）| 配对交易（项目二）|
|----------|------------------|------------------|
| 品种数 | 1 个 | 2 个（rb + hc）|
| 保证金占用 | 1 倍 | 2 倍（同时持有两个品种）|
| 手续费 | 1 倍 | 2 倍（两个品种各收一次）|
| 滑点 | 1 倍 | 2 倍 |
| 腿风险 | 无 | 有（一端成交一端不成交）|
| 最大回撤 | 较高（20-25%）| 较低（8-12%）|
| 年化收益 | 较高（15-20%）| 较低（10-15%）|
| 夏普比率 | 中等（1.0-1.3）| 较高（1.3-1.6）|

**套利策略的数学优势**：你做多一个品种 + 做空一个相关品种，市场整体涨跌对你的影响被对冲了——你只暴露在「价差」这一个风险因子上。这比赌单一品种的涨跌要稳健得多。
:::

## 步骤 2：离线 pandas 回测脚本

```python
import pandas as pd
import numpy as np

df = pd.read_csv('hc_minus_beta_rb.csv', parse_dates=['datetime'], index_col='datetime')
spread = df['spread']
# 滚动均值/标准差（ddof=1，与 m13l1 离线分析口径一致）
mean = spread.rolling(60).mean()
std = spread.rolling(60).std(ddof=1)
zscore = (spread - mean) / std

# 信号：zscore > 2 做空价差（hc 高估 rb 低估），< -2 做多价差
pos = np.where(zscore > 2.0, -1, np.where(zscore < -2.0, 1, np.nan))
pos = pd.Series(pos, index=spread.index).ffill().fillna(0)
# 出场：|zscore| < 0.5
pos = pos.where(zscore.abs() > 0.5).fillna(0)

# PnL = 昨日仓位 × 今日价差变化
pnl = pos.shift(1) * spread.diff()
equity = (1 + pnl / 1e5).cumprod()  # 假设 10 万保证金规模

# 指标
n_years = len(spread) / 252
ann_ret = equity.iloc[-1] ** (1 / n_years) - 1
ann_vol = pnl.std(ddof=1) * np.sqrt(252)
sharpe = ann_ret / ann_vol if ann_vol > 0 else 0
mdd = (equity / equity.cummax() - 1).min()
win = (pnl[pnl != 0] > 0).mean()
print(f'年化收益 {ann_ret:.1%} / 年化波动 {ann_vol:.1%} / 夏普 {sharpe:.2f} / 最大回撤 {mdd:.1%} / 胜率 {win:.0%}')
```
