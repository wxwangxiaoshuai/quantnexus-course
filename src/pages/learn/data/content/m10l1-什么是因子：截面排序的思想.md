到这一节为止，你的策略思维是：「在什么时候做多、什么时候做空」。但职业量化研究有一个更基础的思维框架——**因子思维**。

## 因子的本质：一个截面排序值

因子就是一个数值，它给每一个可交易的品种打一个分，分数高的买、分数低的卖。

最简单的例子：**动量因子**。

- 计算过去 20 天每个品种的涨跌幅
- 涨最多的排第一，跌最多的排最后
- 买入前 5 名（做多），卖出后 5 名（做空）

这一套操作和「何时开仓何时平仓」的策略是两个不同层次的问题：

- **策略回答 WHEN**：今天这个品种是买还是卖？
- **因子回答 WHICH**：在 50 个可交易的品种里，哪些更值得做多？哪些应该做空？

:::highlight blue
因子思维 = 用同一个公式同时评价所有品种，然后买最好的、卖最差的。这比一个一个品种做策略更系统、更可复制——因为你把「选型」和「择时」分离了。
:::

```python
import numpy as np
import pandas as pd

def momentum_factor(prices_dict, lookback=20):
    """
    动量因子：过去N期的收益率
    prices_dict: {symbol: pd.Series of close prices}
    返回: {symbol: factor_value}
    """
    factors = {}
    for symbol, prices in prices_dict.items():
        if len(prices) > lookback:
            ret = prices.iloc[-1] / prices.iloc[-(lookback+1)] - 1
            factors[symbol] = ret
    return factors

def factor_ranking(factors, top_k=5, bottom_k=5):
    """
    按因子值排序，返回做多和做空的品种列表
    """
    ranked = sorted(factors.items(), key=lambda x: x[1], reverse=True)
    longs = [s for s, v in ranked[:top_k]]
    shorts = [s for s, v in ranked[-bottom_k:]]
    print(f"做多({top_k}个): {longs}")
    print(f"做空({bottom_k}个): {shorts}")
    return longs, shorts

# 示例：5个期货品种的动量排序
prices = {
    'rb': pd.Series([3500,3520,3540,3550,3580,3600,3580,3550,3530,3560,3590,
                      3620,3650,3630,3600,3580,3610,3640,3670,3700,3720]),
    'i':  pd.Series([800,805,810,815,820,818,815,820,825,830,835,840,838,
                      842,845,850,855,860,865,870,875]),
    'ma': pd.Series([2500,2480,2460,2440,2420,2450,2480,2500,2520,2500,2480,
                      2460,2440,2420,2400,2380,2400,2420,2400,2380,2360]),
    'ta': pd.Series([5800,5820,5850,5880,5900,5920,5880,5860,5840,5820,5800,
                      5850,5900,5950,5980,6000,6020,6000,5980,5960,6000]),
    'm':  pd.Series([3800,3820,3800,3780,3800,3850,3880,3920,3950,3980,4000,
                      4020,4050,4080,4060,4040,4020,4050,4080,4100,4120]),
}
factor_values = momentum_factor(prices, lookback=5)
factor_ranking(factor_values, top_k=2, bottom_k=2)
```

## 因子四大经典分类（CTA 领域）

**1. 动量因子（Momentum）**：过去涨得好的品种，未来大概率继续涨
- 时间序列动量：品种 A 过去 12 个月的收益率
- 横截面动量：品种 A 相对于其他品种的涨幅排名
- 来自行为金融学的「锚定效应」和「反应不足」

**2. 展期收益（Carry / Roll Yield）**：期货升贴水结构中隐含的收益
- Carry = (近月价 - 远月价) / 近月价
- 升水（Contango）→ 多头换月亏钱 → Carry 为负
- 贴水（Backwardation）→ 多头换月赚钱 → Carry 为正
- **Carry 是期货独有的因子，股票没有**

**3. 价值因子（Value）**：衡量当前价格相对其过去几个月的平均价格是否便宜
- 类似股票里的 PE/PB，期货里用「价格 vs 5年均价」
- 买入便宜的、卖出贵的——长期均值回归

**4. 波动率/质量因子（Low Volatility / Quality）**：低波动品种长期表现优于高波动品种
- 和你直觉相反——「风险低的品种，长期收益并不低」
- 原因：杠杆厌恶者推高了高波动品种的估值

> 这四大因子家族覆盖了 CTA 策略收益的 80%+ 来源。你看到的各种复杂的策略，本质上都是这些因子的包装或组合。

## 因子 vs 策略——关键区分

你之前学习的双均线策略 = 动量因子 + 趋势确认 + 止损规则 的总和。

- **因子是食材**：告诉你哪些品种「新鲜」
- **策略是菜谱**：告诉你怎么把食材做成一道菜（何时买、买多少、怎么止损）

因子研究的意义在于：**先找到有预测力的独立信号来源，再为它设计交易策略**——而不是用一个通不过检验的策略在市场上碰运气。

这就像做软件开发中的单元测试：因子检验（IC、分层回测）就是你的单元测试——写策略之前先跑测试，通过了再继续开发，不通过就直接放弃，节省时间。

:::highlight green
从策略思维到因子思维的升级 = 从「试错」到「系统性搜索」。因子研究不是替代策略，而是在策略之前增加一道质量门——只有通过因子检验的信号才值得加工成策略。
:::
