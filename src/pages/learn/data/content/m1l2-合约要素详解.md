每一个期货合约都有以下核心要素，就像一个 TypeScript 类型定义：

**标的物**：合约交易的是什么（螺纹钢、沪深300指数、原油……）

**合约乘数**：每手合约代表多少实物。螺纹钢 rb2501 每手 = 10 吨。

**最小变动价位（Tick Size）**：价格最小跳动单位。螺纹钢 1 元/吨，即每次报价变化至少 1 元。

**交割月**：合约到期结算的月份。rb2501 = 螺纹钢 2025 年 1 月合约。

```typescript
// 把期货合约想象成 TypeScript 接口
interface FuturesContract {
  symbol: string;          // "rb2501" 螺纹钢2501
  underlying: string;      // "螺纹钢" 标的物
  multiplier: number;      // 10 吨/手
  tickSize: number;        // 1 元/吨
  deliveryMonth: string;   // "2025-01"
  exchange: string;        // "SHFE" 上期所
}

// 价格变动一个 tick，每手盈亏 = tickSize × multiplier
const pnlPerTick = 1 * 10; // = 10 元/手
```

**主力合约**：成交量最大的合约月份，流动性最好，量化策略通常只跑主力合约。当更近的合约到期后，主力会切换到下一个月份，这叫**换月**（roll over）——期货特有的陷阱，我们在数据模块会详细讲。

**合约代码命名规则**（以 rb2501 为例）：

`rb` = 品种代码（螺纹钢）
`25` = 年份（2025 年）
`01` = 月份（1 月）

> 小技巧：品种代码通常是小写英文缩写。i = 铁矿石，j = 焦炭，jm = 焦煤，m = 豆粕，ta = PTA，ma = 甲醇。记住常用品种代码是做期货程序化的基本功。

**完整盈亏计算案例**：

你判断螺纹钢要涨，在 **3500 元/吨** 买入开仓 **1 手**螺纹钢 rb2501。

- 合约乘数：10 吨/手
- 合约价值 = 3500 × 10 = **35,000 元**
- 保证金比例：10%
- 所需保证金 = 35,000 × 10% = **3,500 元**（你实际只需要花 3,500 元就能控制价值 35,000 元的螺纹钢）
- 杠杆倍数 = 35,000 ÷ 3,500 = **10 倍**

**情景 A：价格上涨到 3600 元，你卖出平仓**
- 盈利 = (3600 - 3500) × 10 = **+1,000 元**
- 收益率 = 1,000 ÷ 3,500 = **+28.6%**（价格只涨了 2.86%，但你的本金赚了 28.6%——这就是杠杆！）

**情景 B：价格下跌到 3400 元，你卖出平仓**
- 亏损 = (3400 - 3500) × 10 = **-1,000 元**
- 收益率 = -1,000 ÷ 3,500 = **-28.6%**（价格只跌了 2.86%，你的本金损失了 28.6%）

> 看到没？10 倍杠杆下，价格只动了不到 3%，你的钱已经波动了近 30%。这就是为什么期货被称为「高风险的金融衍生品」。

:::highlight orange
穿仓风险：如果价格连续跌停（跌 5%），你的持仓会损失 5% × 10 倍 = 50%。如果第二天继续跌停，损失可能超过你的保证金，甚至超过账户总资金——这时候经纪商需要你补钱。更极端的情况被称为「穿仓」：你的亏损超过了你的全部本金。这就是为什么保证金制度只是第一道防线，止损纪律才是真正的生命线。
:::

**涨跌停板制度**：国内期货交易所规定每个合约单日的最大涨跌幅（通常为 ±4% 到 ±6%，取决于品种）。涨停板（价格触及涨幅上限）和跌停板（触及跌幅下限）意味着当天价格不能再往这个方向变动更多。

为什么你需要关心这个？
- 你持有的空单如果遇到涨停板，当天可能无法买入平仓，亏损会持续扩大（尤其是封板力度强的时候）
- 涨停板上的交易限制：「涨停板可卖出平仓/卖出开仓（顺势加空），不可买入开仓/买入平仓（不能追涨）；跌停板可买入平仓/买入开仓（顺势加多），不可卖出开仓/卖出平仓（不能杀跌）」

> 日内极端行情下涨跌停封板，是量化策略最大的黑天鹅之一。你的系统必须能识别并处理这种边缘情况。

**交割方式：实物交割 vs 现金交割**：

期货合约到期后有两种结算方式：

- **实物交割**：卖方交付实物商品，买方支付货款。商品期货大多采用此方式（如螺纹钢真的可以收货 10 吨钢筋）。实际中极少发生——大多数持仓会在到期前平仓。

- **现金交割**：按到期日的结算价，盈亏双方现金结算差额。金融期货（股指期货等）和部分商品期货采用此方式。

> 对量化策略的启示：你必须在合约进入交割月前平仓或换仓。如果你忘了，你的策略可能「收到一批螺纹钢」，或者被交易所强制平仓。绝大多数量化系统会在交割月前几个交易日自动移仓。
