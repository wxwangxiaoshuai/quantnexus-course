在模块 1 的开头，你看到了一组数字：螺纹钢现货价格和期货价格。第一次看的人会问一个问题：**为什么同一个东西，现货和期货的价格不一样？**

这个价格差就是**基差（Basis）**——期货量化交易中最重要的概念之一。如果你只学一个关于期货价格的新概念，就学这一个。

:::highlight blue
基差 = 现货价格 - 期货价格

Basis = Spot Price - Futures Price
:::

**为什么会有基差？**

让我们回到面包店老板的故事。他在 3 月份签了一份合约，约定 9 月份以 3000 元/吨的价格买入小麦。

现在是 3 月份，现货小麦价格 2800 元/吨。为什么 9 月合约价格是 3000 元，多了 200？这 200 元是「持仓成本净额」——两个方向相反的力量之差：

1. **持仓成本（Cost of Carry）**：从现在到 9 月份，小麦存在仓库里需要仓储费、保险费、资金占用利息，折合约 250 元。这部分**推动**期货升水（让远月比现货贵）。
2. **便利收益（Convenience Yield）**：手里有现货本身就有价值——工厂可以随时开工、不用担心断供。这个价值约 50 元。便利收益是持仓成本模型里的**减项**——它抵消升水，因为持有现货的便利让你愿意接受更低的远月溢价。

升水 = 持仓成本 − 便利收益 = 250 − 50 = **200**（注意：是「250 − 50 = 200」的相减，而非「150 + 50 = 200」的同向相加——便利收益的方向是抵消升水，不是叠加进升水）。

持有成本模型：$$F = S \cdot e^{(r + storage - cy) \cdot T}$$，其中 $cy$ 为便利收益，是公式里的**减项**。现货越紧缺，便利收益 $cy$ 越高，升水越小甚至转贴水。

基差 = 现货 - 期货 = 2800 - 3000 = **-200**（期货高于现货，这种情况叫「期货升水」）

> 当期货价格 > 现货价格时 → 基差为负 → 期货升水（Contango）。这是大多数商品的常态，因为持仓成本通常大于便利收益。

> 当期货价格 < 现货价格时 → 基差为正 → 期货贴水（Backwardation）。常见于供给紧张时——现货奇缺，便利收益 $cy$ 急剧升高，抵消并反超持仓成本，大家愿意出高价买现货。

:::highlight orange
升贴水结构是期货独有的价格维度——股票不分月份，没有这个概念。理解基差是期货交易的入场券：你不但要判断价格涨跌，还要判断在哪个月份的合约上下注最划算。
:::

**基差如何影响你的量化策略？**

基差不只是定价理论中的概念——它在实战中直接影响策略表现：

**1. 换月盈亏：基差走向决定你的换月是赚是赔**

假设你持有螺纹钢多单，需要从 rb2501 换到 rb2505：
- rb2501 当前价 3400，rb2505 当前价 3450
- 你以 3400 平旧合约，以 3450 开新合约
- 换月「成本」= 3450 - 3400 = 50 元/吨 = 期货升水越大，多头换月亏得越多

相反，如果你是空头：同样升水 50 点 → 在 3400 买回空头、在 3450 新开空头 → 换月赚了 50 元/吨。

> 升水结构（Contango）对空头有利、对多头不利。贴水结构（Backwardation）对多头有利、对空头不利。这个不对称性直接影响长期策略的收益——你的策略跑赢需要覆盖这个「展期成本」。

**2. 基差率作为择时信号**

当基差率异常扩大（超过历史 95% 分位数）时，可能是极端行情的信号：期货升水极端扩大 → 市场对未来预期过度乐观；期货贴水极端扩大 → 现货紧缺到恐慌程度。

**3. 基差影响策略回测**

复权连续合约的回测中，基差是隐含在价格里的。如果策略在回测中很赚钱，但遇到极端升水年份（多头换月持续出血），实盘可能大打折扣。

```python
# 基差与展期收益的实战计算

def calc_basis_rate(futures_price, spot_price):
    """基差率 = (现货 - 期货) / 现货"""
    return (spot_price - futures_price) / spot_price

def calc_roll_yield(front_price, next_price, position='long'):
    """展期收益率：换到下月合约的隐含成本/收益"""
    roll_return = (front_price - next_price) / front_price
    sign = 1 if position == 'long' else -1
    return sign * roll_return

# 案例：升水结构下多头 vs 空头
front = 3400  # 近月合约
next_m = 3450  # 远月合约

print(f"多头展期收益: {calc_roll_yield(front, next_m, 'long'):.2%}")
# → 多头展期收益: -1.47% (每换月亏1.47%)
print(f"空头展期收益: {calc_roll_yield(front, next_m, 'short'):.2%}")
# → 空头展期收益: +1.47% (每换月赚1.47%)
```
